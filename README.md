# neural-network-backpropagation-from-scratch

This is a very small neural network that can be trained on a small set of data to multiply numbers by 2 with decent accuracy. The math behind finding the gradient using backpropagation is derived carefully and there is no use of ML libraries.

Why did I build this?
I wanted to understand how neural networks work by breaking them down into their most basic components. 
Rather than treating the neural network as a "black box" that just works, I wanted to understand what is actually going on mathematically that allows neural networks to function.
